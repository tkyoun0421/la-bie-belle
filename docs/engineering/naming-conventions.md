# Naming Conventions

- General folders and files use `kebab-case`.
- React component files use `PascalCase.tsx`.
- Hook files use `use-*.ts`.
- Utility, library, service, and repository files use `camelCase.ts`.
- Test files use `*.test.ts` or `*.test.tsx`.
- Database migration files use `YYYYMMDDHHMMSS_description.sql`.
- Environment variables use `UPPER_SNAKE_CASE`.
- AI harness artifacts keep the existing names, such as `task-spec.md`, `plan.md`, and `spec.md`.
