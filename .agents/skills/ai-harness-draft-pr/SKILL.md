# AI 하네스 초안 PR

초안 PR 단계만 수행한다.

## 선행 조건

- `review-score.json`의 decision이 `PASS`여야 한다.
- `validate-score.mjs` 검증을 통과해야 한다.

## 절차

1. 원본 이슈, 작업 명세, 계획, 구현 기록, 검증 기록, 리뷰 기록을 읽는다.
2. 변경 범위를 확인하고 커밋 메시지와 PR 본문을 작성한다.
3. 초안 PR을 생성한다.
4. 이슈/PR 요약에는 목표, 변경 요약, 검증 결과, 리뷰어 점수, 남은 리스크를 포함한다.

## 금지

- `REWORK` 또는 `FAIL` 상태에서 초안 PR을 만들지 않는다.
- PR Ready 전환, 머지, 배포를 하지 않는다.
